# AR Dare Lab

Native SwiftUI MVP for a local-first AR party challenge game.

- iOS 17+
- SwiftUI, SwiftData, ARKit, RealityKit, Vision, StoreKit 2, ReplayKit
- Simulator-safe demo mode when AR is unavailable
- Challenge engine for Laser Dodge, Lava Jump, Freeze Pose, Portal Hunt, and Crown Balance

Open `codex_app_2.xcodeproj` in Xcode and run the `ARDareLab` scheme.
