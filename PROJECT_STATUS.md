# AR Dare Lab project status

A complete SwiftUI / ARKit MVP has been generated for this repo as a local project package in the ChatGPT workspace.

Included in the generated package:

- `codex_app_2.xcodeproj`
- SwiftUI app shell with Home, Play, Packs, Leaderboard, Settings
- ARKit / RealityKit arena surface with simulator fallback
- Vision body-pose service scaffold
- Game engine and rules for Laser Dodge, Lava Jump, Freeze Pose, Portal Hunt, Crown Balance
- SwiftData local models
- StoreKit 2 product IDs and entitlement handling
- ReplayKit recording hooks and scorecard export fallback
- Unit tests for scoring, rules, and state transitions

Open the downloaded project package in Xcode and run the `ARDareLab` scheme on an iOS 17+ simulator or device.
